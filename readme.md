## Installing this app
- Create a database 
- Copy .env.example as .env
- Edit the database name and auth in .env file
- Run (composer install) command
- Run (php artisan key:generate)
- Run (php artisan migrate)
- Run (php artisan db:seed)
- Finally (php artisan serve)
